# Cloud Storage

AWS
- S3 buckets
- EFS (Fileserver, no VM)
- EBS (EC2 virtual drive storage)

Azure
- Storage account (not for VMs)
- Managed disks (for VMs)

Google Cloud
- Storage buckets (like S3)
- Compute engine disks and images (like EFS/EBS)
